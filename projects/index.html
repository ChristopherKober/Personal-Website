NYC Exploration Map

Interactive mapping logging visits by neighborhood

This project uses an angular front-end to display all of the neighborhoods in Manhattan, along with timeline data for each neighborhood. This project follows the travels of an NYC native over the course of time. The front end is powered by a node JS Rest Api and saves its data in a PostgreSQL database.
